# Remote Management UI

Mobile-friendly interface for operators to manage LiveShow content and control the On-Stage display.

## Responsibilities

- Allow operators to submit and manage Scripts and ScriptQueues from any mobile device
- Provide remote control of the On-Stage teleprompter view

## Features

- **Script management**
  - Post a new Script (title, body, submitted by)
- **ScriptQueue management**
  - Create a new ScriptQueue
  - Edit / update an existing ScriptQueue
  - Delete a ScriptQueue
- **On-Stage view controls**
  - Select the active Script
  - Page Up / Page Down
  - Configure auto-scroll speed
