# UI

This directory contains the web-based user interface components for LiveShow.

## Sub-components

| Directory | Description |
|-----------|-------------|
| `on-stage/` | Full-screen 10-foot teleprompter display used during live performances |
| `remote-management/` | Mobile-friendly interface for operators to manage scripts and queues |

Both sub-components communicate with the **API** component over HTTP and are independently deployable.
