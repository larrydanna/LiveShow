# On-Stage UI

Full-screen 10-foot teleprompter display for live stage performances.

## Responsibilities

- Display a selected Script from a ScriptQueue in a large, readable format
- Provide teleprompter controls for the performer or operator at the venue

## Features

- Select a ScriptQueue
- Select a Script within the queue
- Teleprompter view
  - Live performance script display (large text, high contrast)
  - Manual paging controls (Page Up / Page Down)
  - Automatic scrolling with speed adjustment
  - Keyboard control
  - Escape back to ScriptQueue selection
